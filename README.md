# Reconstruction Tax Calculator

Sales tool for Yavardi to quantify the financial impact of missing digital proof of authority in law firms.

## Deploy in 5 Steps

### 1. Push to GitHub
```bash
cd reconstruction-tax-calculator
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR-USERNAME/reconstruction-tax-calculator.git
git push -u origin main
```

### 2. Deploy to Vercel
- Go to vercel.com
- Sign up with GitHub
- Click "Add New Project"
- Import your repo
- Click "Deploy"

Done in ~2 minutes. You'll get a URL like `reconstruction-tax-calculator.vercel.app`

### 3. Test
- Open the URL
- Input test numbers
- Verify calculations work
- Bookmark it

### 4. Share with Team
Send Paul and Tracie:
- The live URL
- When to use: during discovery after pulling firm metrics
- Bookmark on all devices

### 5. (Optional) Custom Domain
In Vercel: Settings → Domains → Add `calculator.yavardi.com`

## Local Development
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Updates
```bash
git add .
git commit -m "Update"
git push
```
Vercel auto-deploys in ~60 seconds.
